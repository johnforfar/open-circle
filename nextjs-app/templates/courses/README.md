# Course Templates

This directory contains course templates based on official blockchain education curricula.

## Argentina Blockchain Curriculum

The `argentina-blockchain-curriculum.json` file contains the complete course structure based on the official blockchain curriculum introduced in Buenos Aires high schools, in collaboration with the ETH Kipu Foundation.

### Curriculum Overview

The curriculum is designed for:
- **High school students** (ages 16-18)
- **Adult learners** (18+)

### Course Structure

1. **Introduction to Blockchain Technology** (8 weeks)
   - Fundamentals of blockchain
   - Applications and use cases
   - Architecture and security

2. **Ethereum Platform Overview** (10 weeks)
   - Ethereum architecture
   - Smart contracts
   - Ethereum ecosystem

3. **Solidity Programming** (12 weeks)
   - Solidity basics and advanced concepts
   - Smart contract development
   - DApp development

4. **Blockchain Internships** (16 weeks)
   - Real-world project experience
   - Industry mentorship
   - Portfolio development

5. **Ethical and Legal Considerations** (6 weeks)
   - Blockchain ethics
   - Legal framework
   - Responsible development

### Usage

This template can be imported into the Open Circle platform to create courses. The structure includes:

- Detailed module breakdowns
- Learning outcomes
- Assessment methods
- Prerequisites
- Duration estimates
- Practical projects

### Implementation Notes

- All courses are designed to be Web3-native
- Wallet connection is required for practical exercises
- Email collection is integrated for course enrollment
- Community features support peer learning and collaboration
